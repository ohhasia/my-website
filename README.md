## 📋 项目简介

**农链通** 是一个功能完整的农产品交易服务平台，采用商城样式设计，提供从浏览到购买的完整购物体验。平台涵盖农产品批发、行情数据、交易培训、仓单质押、助贷服务等多个功能模块。

**项目特色：**
- 🎨 商城样式首页设计
- 📦 12款精选农产品（200-500元）
- 🔍 搜索、筛选、排序功能
- 🛒 购物车和结算功能
- 📱 响应式设计，完美适配多端
- 📊 专业数据可视化
- 💰 金融服务支持

## 🔧 常见问题

**Q1：页面无法访问？**
A1：检查服务器是否正常运行，确保端口8888未被占用

**Q2：样式显示异常？**
A2：清除浏览器缓存，重新加载页面

**Q3：搜索功能不工作？**
A3：检查浏览器控制台是否有错误，确保app.js正确加载

**Q4：购物车数据丢失？**
A4：购物车数据存储在localStorage，清除浏览器数据会导致丢失

**Q5：商品图片不显示？**
A5：商品使用emoji图标，确保浏览器支持emoji显示
